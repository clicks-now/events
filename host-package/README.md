# Clicks Host Package

Everything you need to run a Clicks event in your city.

---

## What you need

**Venue**
- Capacity for 40–80 people, standing or mixed seating
- AV: screen + HDMI, handheld mic optional
- Bar or drinks service (sponsors can cover costs)
- Central location, easy transit access

**Co-host partner**
- One Shopify ecosystem partner to co-host with you
- They help promote, split logistics, and co-brand the event
- They should not dominate the agenda — this is a community event, not a launch party

**Speakers**
- 3–4 speed round speakers (5 min each, hard cut)
- Topics: what you're building, what's working, what broke
- No pitches. No decks required.

**Facilitator**
- One person who runs the room
- Keeps time, introduces speakers, opens/closes
- Can be you or a co-host

---

## Format

| Time | Segment |
|------|---------|
| 0:00 | Doors open, drinks |
| 0:30 | Welcome + housekeeping (5 min) |
| 0:35 | Speed rounds (5 min × 4 speakers) |
| 1:05 | Break + networking |
| 1:25 | Open floor / facilitated discussion (optional) |
| 1:50 | Closing remarks |
| 2:00 | Hard out |

2 hours. Hard out. No exceptions — it sets the tone.

---

## Before the event

- [ ] Venue confirmed + deposit paid
- [ ] Co-host partner confirmed
- [ ] Date and city listed (open an issue in this repo)
- [ ] Speakers confirmed (3–4)
- [ ] Event page live (clicks.now or Luma)
- [ ] Promoted via Discord + social
- [ ] Run-of-show reviewed with co-host

## Day of

- [ ] Arrive 45 min early
- [ ] Test AV
- [ ] Brief speakers on timing (5 min, hard cut)
- [ ] Have someone on the door
- [ ] Keep the 2-hour hard out

## After the event

- [ ] Send thank-you to speakers and co-host
- [ ] Post photos to Discord
- [ ] Submit event feedback PR (use template in `templates/`)
- [ ] Share any lessons back to this repo

---

## Co-branding guidelines

- Event is always **Clicks [City]** — not your company's event
- Co-host logo can appear alongside Clicks branding
- No sponsor logos on the stage or podium
- Sponsors acknowledged verbally at open/close only
